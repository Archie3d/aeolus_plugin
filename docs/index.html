Aeolus

Aeolus is a pipe organ synthesizer (VST plugin) inspired by the original Aeolous project.

The present implementation follows the pipes audio generation of the original Aeolous, but introduces a number of changes and improvements:

  • Pipe models can be composed of multiple pipes (to create proper mixtures);
  • Different pipes can be used across the keyboard compass (this enables mixtures breaking-back);
  • Pipes chiff noise is played on attack;
  • The plugin uses zero delay convolutional reverb with selectable IRs from OpenAir library;
  • Registration presets also acts as a sequencer.
